## ⚡ Overview

**CK-Deface** is a single-file Python CLI tool for generating fully animated, professionally styled deface pages with a hacker-grade terminal experience. No pip installs. No extra folders. Just drop the file and run.

> Originally built by **CyberKallan** — Completely recoded and upgraded to v2.0 by **Hypervenom**

```
┌─────────────────────────────────────────────────────────────────┐
│  One .py file  ·  10 themes  ·  6 fonts  ·  Live preview        │
│  YouTube / MP3 / SoundCloud auto-embed  ·  SHA-256 encryption   │
│  Download server  ·  Mobile mode  ·  Profile save / load        │
└─────────────────────────────────────────────────────────────────┘
```

<details>
<summary><b>🎵 YouTube / MP3 / SoundCloud — Auto-Embed Engine</b></summary>
<br/>

Paste any media URL. The tool detects the source and injects the correct embed — zero manual config.

| URL Pasted | Detected As | What Gets Generated |
|---|---|---|
| `youtube.com/watch?v=ID` | YouTube | Invisible iframe — plays on first click |
| `youtu.be/ID` | YouTube | Same |
| `youtube.com/shorts/ID` | YouTube | Same |
| `example.com/song.mp3` | Direct MP3 | `<audio>` tag — unlocks on click |
| `soundcloud.com/artist/track` | SoundCloud | Hidden SC widget — autoplays on click |
| *(blank)* | None | No audio injected |

> **Why click-to-play?** All modern browsers block autoplay until a user interacts with the page. The tool handles this automatically — media unlocks the moment the visitor clicks anywhere.

</details>

<details>
<summary><b>👁 Live Preview Server — localhost:7070</b></summary>
<br/>

After generating a page, pick **Option 3** from the main menu. A local HTTP server starts instantly and opens your default browser to preview the page exactly as visitors will see it.

```
✔ Preview → http://localhost:7070
  Press Enter to stop the server...
```

No extra software needed. Pure Python stdlib `http.server`.

</details>

<details>
<summary><b>📥 Animated Download Page — localhost:8080</b></summary>
<br/>

Pick **Option 4** to launch a dedicated hacker-style download page for your generated file.

**Download page includes:**
- Live matrix rain canvas background
- Animated progress bar with status messages (`Compiling assets... → Ready!`)
- File metadata cards (filename, size, theme, font)
- Click spark effects on every interaction
- One-click `index.html` download button that appears after the progress animation

```
✔ Download page → http://localhost:8080
  Press Enter to stop the server...
```

</details>

<details>
<summary><b>🔐 SHA-256 Page Encryption</b></summary>
<br/>

Enable encryption in the wizard and set a password. The generated HTML is base64-encoded and wrapped in a password gate page.

**How it works:**
1. Visitor opens the page — sees a fake `403 Forbidden` terminal screen
2. They enter the password — verified client-side via `SubtleCrypto.digest('SHA-256')`
3. Wrong password → error shown, input cleared, page stays locked
4. Correct password → real deface page loads instantly, no page reload

**No server required.** All logic runs in the visitor's browser.

```
┌───────────────────────────────────┐
│  🔐  ACCESS DENIED                │
│  This page is protected.          │
│                                   │
│  [ _ _ _ _ _ _ _ _ _ _ _ _ ]     │
│         AUTHENTICATE              │
│                                   │
│  ⚠ Incorrect key. Access denied.  │
└───────────────────────────────────┘
```

</details>

<details>
<summary><b>📱 Mobile-First Responsive Mode</b></summary>
<br/>

Toggle this on during page creation to inject extra CSS optimized for phones:

- Fluid `clamp()` font sizing for all screen widths
- Compact padding and element spacing
- Touch-friendly button tap targets
- Hover transitions disabled on touch devices (`@media(hover:none)`)

Works on top of any theme — fully compatible.

</details>

<details>
<summary><b>💾 Profile Auto-Save / Load</b></summary>
<br/>

Every time you generate a page, your full config is written to `profiles/last.json` automatically. On your next session, select **Option 2** from the menu to reload it — all fields pre-filled, ready to tweak or regenerate instantly.

</details>

## 📥 Installation & Usage

### Step 1 — Clone

**Termux / Linux / macOS**
```bash
pkg update && pkg upgrade -y
pkg install python git -y

git clone https://github.com/cyberkallan/ckdeface.git
cd ckdeface
```

**Windows (PowerShell)**
```powershell
git clone https://github.com/cyberkallan/ckdeface.git
cd ckdeface
```

### Step 2 — Run

```bash
python3 hagepev4.py
```

> ✅ **Python 3.9+** &nbsp;·&nbsp; ✅ **No pip installs** &nbsp;·&nbsp; ✅ **Pure stdlib only**
